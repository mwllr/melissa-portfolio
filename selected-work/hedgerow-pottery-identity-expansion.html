Hedgerow Pottery Identity Expansion

Expanding a single logo into a flexible identity system that could support digital, print, apparel, and physical signage.

Challenge

The client had an established pottery business but only possessed a small, low-resolution logo file that limited how the brand could be used.

Several challenges emerged:

  • The existing logo did not scale well for large-format applications.
  • File quality was insufficient for signage and print production.
  • Color combinations created accessibility and readability concerns.
  • The logo included descriptive text that became difficult to read at smaller sizes.
  • No supporting brand assets or guidelines existed.

As the business grew, the client needed a more flexible visual identity that could be used consistently across a variety of applications.

Approach

Rather than simply recreating the existing logo, I approached the project as a brand system expansion.

The goal was to preserve the recognizable elements of the existing identity while creating a collection of assets that would work across both digital and physical environments.

Special attention was given to scalability, readability, accessibility, print production requirements, and long-term maintainability.

The result was a flexible identity system that could support everything from social media graphics to exterior signage.

What Was Created

Expanded Logo System

The original logo was rebuilt as a professional vector graphic, allowing it to scale without losing quality.

Multiple logo variations were created to support different use cases, including primary logo, simplified logo variations, horizontal layouts, vertical layouts, single-color versions, and high-contrast versions.

This provided flexibility while maintaining a consistent brand presence.

Brand Standards

To help the client maintain consistency over time, I created a lightweight style guide that documented brand colors, typography, logo usage recommendations, file specifications, and application examples.

The guide was designed to be practical and approachable for a small business owner rather than overly formal.

Accessibility Improvements

The redesign improved readability and usability by simplifying visual elements, improving color contrast, reducing unnecessary text within the logo, creating versions suitable for different backgrounds, and increasing legibility at smaller sizes.

These improvements helped ensure the brand remained recognizable and functional across a wider range of situations.

Client Education

One challenge many small businesses face is understanding when to use different file types.

To support future success, I provided educational materials explaining vector vs. raster graphics, print file requirements, web graphics, signage specifications, and appropriate use cases for each file format.

This helped the client work more effectively with printers, vendors, and future designers.

Real-World Applications

The expanded identity system was successfully applied across multiple touchpoints.

For apparel, it supported branded t-shirts, promotional merchandise, and event apparel. For digital use, it supported website graphics, social media assets, and marketing materials. For physical signage, it supported exterior building signage, large-format graphics, and print collateral.

One significant benefit was the ability to produce large exterior signage without the pixelation and quality issues that would have occurred using the original logo file.

Outcomes

The project transformed a single low-resolution logo into a scalable identity system that could grow alongside the business.

Benefits included:

  • Improved brand consistency
  • Greater flexibility across mediums
  • Better readability and accessibility
  • Professional production-ready assets
  • Reduced future design limitations
  • Increased confidence when working with vendors and printers

Key Takeaways

While the project was smaller in scale than many of my enterprise initiatives, it reinforced a lesson that applies to organizations of every size:

Good systems create flexibility.

Whether building a university-wide design system or a small business identity system, success comes from creating clear standards, reusable assets, and practical guidance that helps others use them effectively.
